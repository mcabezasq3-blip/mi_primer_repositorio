# Informe del Repositorio

**Archivos incluidos:**  
- `datos_ventas.csv`  
- `datos_clientes.csv`  

---

### `datos_clientes.csv`  
**Contenido:** 15 clientes ficticios de Ecuador.  
**Columnas:**  
- `Nombre`  
- `Fecha_Nacimiento` (YYYY-MM-DD)  
- `CI` (formato `09XXXXXXXX` con dígito verificador válido)  

**Uso:** Pruebas de sistemas, validación de cédulas.

---

### `datos_ventas.csv`  
**Contenido:** 15 registros de ventas mensuales.  
**Columnas:**  
- `Mes` (YYYY-MM)  
- `Ventas` (unidades)  
- `Monto_Total` (USD)  

**Uso:** Análisis de ventas, reportes o dashboards.

---

**Datos:** 100% ficticios y generados aleatoriamente.  
**Formatos:** Compatibles con Excel, Google Sheets, Python, etc.
